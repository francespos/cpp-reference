# program
Here is the C++/CMake template for programs.
## How to build
```sh
$ cd build
$ cmake -S .. -B .
$ cmake --build .
```
### How to use git
Il commit salva i cambiamenti localmente, il push spinge i commit locali nel
server remoto (es. Github).
```sh
$ git add .
$ git commit -m "."
$ git push origin main
```
